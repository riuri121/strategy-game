# Strategy Game

A strategy game project built with Python and Rust.

## Setup

### Python

1. Clone this repository
2. Install dependencies: `pip install -r python/requirements.txt`
3. Run development server: `python python/main.py`

### Rust

1. Navigate to the Rust project: `cd rust/`
2. Build: `cargo build`
3. Run: `cargo run`

## Project Structure

- `python/` - Core game logic and utilities
- `rust/` - Performance-critical components and systems
- `design/` - Game design documents
- `tests/` - Test suites for both Python and Rust

## Branches

- **main** - Production-ready releases
- **dev** - Active development
