# Rust Development

Performance-critical components and systems written in Rust.

## Setup

Ensure you have Rust installed: https://www.rust-lang.org/tools/install

## Building

```bash
cargo build
```

For release builds:

```bash
cargo build --release
```

## Running

```bash
cargo run
```

## Testing

```bash
cargo test
```

## Formatting & Linting

```bash
# Format code
cargo fmt

# Check code
cargo clippy
```

## Structure

- `src/main.rs` - Entry point
- `Cargo.toml` - Dependencies and project metadata
